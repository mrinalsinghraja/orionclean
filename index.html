OrionClean was designed with privacy as a first principle — not an afterthought. Every architectural decision reflects this.

🚫NO DATA COLLECTION
👤NO ACCOUNTS REQUIRED
🤖ON-DEVICE AI ONLY
📍NO TRACKING
💾LOCAL STORAGE ONLY
🔐APP SANDBOX
📂USER-GRANTED ACCESS ONLY
✅MAC APP STORE APPROVED

OrionClean operates under App Sandbox — the strictest security model for macOS apps. It only accesses folders you explicitly grant permission to via the standard macOS file picker. No Full Disk Access is requested or required. The AI runs entirely on your device via Apple's FoundationModels framework with no internet connection. All analytics, history, and achievement data are stored locally using Apple's UserDefaults — never transmitted externally.

// frequently asked questions
Is OrionClean safe to use?
Yes. Every file receives an AI safety assessment — Safe, Review, or Risky — before you can delete it. OrionClean never automatically deletes anything. You review and confirm every cleanup action.
What macOS version do I need?
OrionClean requires macOS 14 Sonoma or later. Full AI features (Apple Intelligence) require macOS 15.1+ on an Apple Silicon Mac with Apple Intelligence enabled. On older systems, OrionClean automatically falls back to intelligent heuristic analysis — all features remain available.
Does OrionClean need Full Disk Access?
No. OrionClean does not request or require Full Disk Access. It uses standard macOS user-granted folder access — when a scan needs access to a folder, it shows the standard macOS file picker and you choose which folders to grant. You stay in full control at all times.
Is it really free? What's the catch?
Genuinely free. No subscription, no premium tier, no time-limited trial. Every feature is available from day one. OrionClean is a free launch — we believe in building an audience first and earning trust before ever charging anything.
Does OrionClean send any data to the internet?
The only network activity is the Network Quality feature, which uses standard outbound connections to measure your internet speed and latency — similar to how a speed test works. No personal data, file names, or system information is ever transmitted. All other app activity is completely local.
Can OrionClean uninstall apps?
The App Store version of OrionClean does not include an app uninstaller — this feature requires deeper system access that isn't compatible with Mac App Store sandbox rules. OrionClean focuses on file-level cleanup, storage analysis, performance monitoring, and AI diagnostics.
How do I get the best results from a scan?
When OrionClean prompts you to grant folder access, add your home folder (~/) and any external drives you want scanned. The more folders you grant access to, the more comprehensively OrionClean can find duplicates, large files, and junk. You can manage granted folders at any time in Settings.
